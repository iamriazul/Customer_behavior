
## Project Overview

This project analyzes customer shopping behavior using **Python, PostgreSQL, SQL, and Power BI** to uncover valuable business insights. The workflow includes data cleaning, feature engineering, database integration, SQL-based business analysis, and interactive dashboard creation.

The objective is to help businesses better understand customer purchasing patterns, spending habits, product preferences, and customer demographics to support data-driven decision-making.

## Tools & Technologies

* **Python** (Pandas, SQLAlchemy)
* **Jupyter Notebook**
* **PostgreSQL**
* **SQL**
* **Power BI**

## Project Workflow

### 1. Data Cleaning & Preprocessing (Python)

* Loaded the customer shopping behavior dataset using Pandas.
* Performed exploratory data analysis (EDA).
* Checked for missing values and handled null values.
* Filled missing review ratings using the median rating of each product category.
* Renamed columns using snake_case for better readability.
* Created new features such as:

  * **Age Group**
  * **Purchase Frequency (Days)**
* Removed redundant columns after validation.
* Prepared the cleaned dataset for database storage.

### 2. Database Integration (PostgreSQL)

* Connected Python with PostgreSQL using SQLAlchemy.
* Imported the cleaned dataset into PostgreSQL.
* Structured the data for efficient SQL querying and business analysis.

### 3. Business Analysis Using SQL

Performed analytical SQL queries to answer important business questions, including:

* Revenue comparison between male and female customers.
* Customers who used discounts but still spent above the average purchase amount.
* Top-rated products based on customer reviews.
* Purchase comparison between Standard and Express shipping.
* Spending behavior of subscribed vs. non-subscribed customers.
* Additional customer behavior and sales insights using aggregate functions, filtering, grouping, and subqueries.

### 4. Interactive Dashboard (Power BI)

Built an interactive Power BI dashboard to visualize customer behavior and business performance.

The dashboard provides insights into:

* Customer demographics
* Revenue analysis
* Product performance
* Purchase trends
* Customer segmentation
* Shopping behavior patterns
* Interactive filters and slicers for dynamic analysis
